# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-03

### Added
- ✨ Initial project setup with React 18 + TypeScript + Vite
- 🎨 Tailwind CSS integration with custom theme
- 🧩 Shadcn UI components (Button, Input, Card, Dialog, Select, Textarea, Label, Badge, Separator)
- 📑 Complete routing with React Router v6
- 👥 Client management system (Consumer & Company types)
- 📄 Quote editor with product management
- 💾 LocalStorage API for data persistence
- 📊 Dashboard with statistics
- 📋 Quotes listing and management
- 📅 History view with full tracking
- ⚙️ Settings page for company configuration
- 🎯 PDF generation with jsPDF + html2canvas
- 📱 Responsive design (Mobile, Tablet, Desktop)
- 🌍 Spanish (El Salvador) localization
- 💱 Currency formatting (USD)
- 🔢 Automatic IVA calculation (13%)
- 🎨 Professional design inspired by Apple, Stripe, Linear, Notion
- 📋 Complete TypeScript type definitions
- 🛡️ Data validation and sanitization
- 🚀 Performance optimized

### Features
- Dashboard with KPIs (Total Quotes, Clients, Monthly Stats, Revenue)
- Create/Edit clients with role-based forms
- Create/Edit quotations with inline product editor
- Automatic quote numbering
- PDF export functionality
- Company settings management
- Bank account configuration
- Default text templates
- Color customization
- Responsive sidebar navigation
- Real-time calculations
- Status tracking (Draft, Sent, Accepted, Rejected)

### Technical
- React 18.2.0
- TypeScript 5.2.2
- Vite 5.0.8
- Tailwind CSS 3.3.6
- Radix UI components
- Lucide Icons
- React Router 6.20.0
- UUID for ID generation
- jsPDF for PDF generation
- html2canvas for DOM capture
